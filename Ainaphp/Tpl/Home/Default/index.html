Way of working

Safe Shield test process may vary from country to country. Here take Australia as an example.

1.Check the piping system to see whether there is any leakage at the welding points, valves and plugs, then install the Safe Shield device

2.Login the APP and fill in the form

3.Pressure test stage: Select pressure test group, pressurize the system to setting pressure then start test. The hydrostatic test has one round which lasts 30 mins.

4.When the pressure testing is completed, it will form a report and clicked to send a PDF file to tester's mailbox.

Pressure test process can be customized.
